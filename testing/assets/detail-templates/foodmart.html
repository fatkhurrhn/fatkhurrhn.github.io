FoodMart is an eCommerce website template for online grocery store services. It uses modern technologies like Bootstrap 5, HTML5, and CSS3. The design has a simple outlook with all the necessary features for a perfect online grocery store. The free template includes several responsive UIs. There is a search filter with a dropdown menu bar presenting all the categories, which ensures a smooth search across the pages. The top navigation bar has an additional dropdown menu bar for enhanced navigation across the pages. Scrolling down, there are banners, category cards, category sliders, product grids, tabbed contents, email subscription UI, and newsletter subscription form UI. The mobile-first design also incorporates a preloader, on-hover effects, call-to-action buttons, and more to ensure the best services. Moreover, this theme is responsive and runs smoothly on small to large screens without hassle
